How I Built My First Website

Published on March 25, 2025

Creating a website from scratch was both exciting and challenging. Here is what I learned from the experience (and what I’d do differently next time!).

Why I Decided to Build a Website

I wanted to learn web development and practice my HTML & CSS skills. This was my first project, and I was eager to see how everything comes together—from structuring a webpage to styling it.

What I Learned

  • How to structure an HTML document properly.
  • The importance of using external CSS for better styling control.
  • How colors, fonts, and spacing affect the design of a webpage.
  • Debugging mistakes and testing in different browsers is crucial.

What I Would Do Differently

While I’m happy with how this project turned out, there are a few things I’d improve in the next one:

  • Use better design principles for a more visually appealing look.
  • Make the website responsive so it looks good on all devices.
  • Experiment with JavaScript to add interactive elements.
  • Host it online using GitHub Pages so others can see it.

What's Next?

Now that I have built my first website, I plan to continue learning more about CSS frameworks like Bootstrap, and eventually dive into JavaScript to make my websites more interactive. Stay tuned for my next project!
